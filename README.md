# VibeCoded Workout

A simple, client-side workout application that runs entirely in the browser. Perfect for quiet, equipment-free workouts at home.

## Features

*   **Workout Generator:** Create custom workouts based on duration, difficulty, and target body parts.
*   **Workout Player:** Follow along with a timer, exercise images, and descriptions.
*   **Progress Tracking:** Save your workout history and visualize your progress over time.
*   **No Backend Required:** Runs entirely in the browser using HTML, CSS, and JavaScript.
*   **GitHub Pages Ready:** Can be easily deployed to GitHub Pages.

## How to Use

1.  Clone this repository.
2.  Open `index.html` in your web browser.

That's it! No build steps or dependencies required.

## How to Deploy to GitHub Pages

1.  Push your code to a GitHub repository.
2.  In your repository settings, go to the "Pages" section.
3.  Select the branch you want to deploy from (usually `main` or `master`).
4.  Choose the `/(root)` directory for the source.
5.  Click "Save". Your site will be deployed to `https://<your-username>.github.io/<your-repository-name>/`.
